üöÄ Rezi SDK Demo

Welcome to the Rezi SDK demonstration! This SDK provides easy-to-use, beautiful components that you can integrate into any no-code website tool.

üí° For No-Code Tools: Simply include the CSS and JS files in your website's head section, then use the simple JavaScript API to create components. No build process required!

üì± Buttons

Create beautiful, responsive buttons with different styles and sizes:

// Create buttons easily const primaryBtn = ReziSDK.createButton({ text: 'Primary Button', variant: 'primary', size: 'md', onClick: () => alert('Primary clicked!') }); primaryBtn.render('#button-container'); const secondaryBtn = ReziSDK.createButton({ text: 'Secondary', variant: 'secondary', size: 'sm' }); secondaryBtn.render('#button-container');

üçû Toast Notifications

Show elegant toast notifications for user feedback:

// Show different types of toasts ReziSDK.showToast({ message: 'Success! Your action was completed.', type: 'success', duration: 3000, position: 'top-right' }); ReziSDK.showToast({ message: 'Error: Something went wrong.', type: 'error', position: 'top-center' });

üîß Integration Guide

To use Rezi SDK in your no-code website tool:

1. Include the CSS and JS files:

<!-- Add to your HTML head --> <link rel="stylesheet" href="path/to/rezi-sdk.css"> <script src="path/to/rezi-sdk.min.js"></script>

2. Use the components:

// The SDK is available globally as ReziSDK const button = ReziSDK.createButton({ text: 'Click me!', variant: 'primary', onClick: () => console.log('Button clicked!') }); // Render to any container button.render('#my-container'); // Show notifications ReziSDK.showToast({ message: 'Hello from Rezi SDK!', type: 'success' });

3. Customize themes:

// You can customize colors via CSS variables :root { --rezi-primary: #your-brand-color; --rezi-success: #your-success-color; } // Or pass theme in component config const darkButton = ReziSDK.createButton({ text: 'Dark Button', theme: 'dark', variant: 'primary' });

üìö Quick API Reference

ReziSDK.createButton(config)

{ text: string, // Button text variant: 'primary' | 'secondary' | 'outline', size: 'sm' | 'md' | 'lg', theme: 'light' | 'dark', onClick: function, // Click handler disabled: boolean }

ReziSDK.showToast(config)

{ message: string, // Toast message type: 'success' | 'error' | 'warning' | 'info', duration: number, // Auto-hide duration (ms) position: 'top-right' | 'top-left' | 'bottom-right' | 'bottom-left' }
